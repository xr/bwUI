    Description

    class-darken

    @mixin class-darken($prefix, $property) { ... }

    Description

    This mixin can be used to darken with specific prefix this can be useful when we want to modify the state of with the same class name. change state but keep the same class name.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $prefix

    in case we want to modify background, we have to specify prefix

    		Prefix— none
    $property

    the css property we want to change

    		Property— none

    Example

    set the default and modifiers to selector .header.

    .button {
  &:hover {
      @include class-darken();
  }
}
