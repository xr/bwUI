    font-size-modifier

    @mixin font-size-modifier($name) { ... }

    Description

    This mixin can be used to set the default font-size based on $font-sizes map and attach the font-size specific modifiers to elements or classes based on $font-sizes-variations map.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $name

    the property name from $font-sizes map.

    		Name— none

    Example

    set the default and modifiers to selector .header.

    .header {
  @include font-size-modifier(header);
}
